One-time setup

This messenger uses a free Supabase project as its shared backend — the same one your Task Board uses.

  1. Open index.html and paste your Supabase Project URL and anon public key into the CONFIG block near the bottom (identical to the Task Board values).
  2. Make sure the users and messages tables exist (see README.md).
  3. Commit and push. Everyone with the link shares the same buddy list and messages.
